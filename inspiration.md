## How this file gets used (the process — don't skip)

1. **Lock this file first.** All 6 sections filled; readable out loud in ~90 seconds;
   2–3 non-web references saved in `inspiration/references/`.
2. **Design SYSTEM before screens.** Feed this file to Claude Design and generate the
   *system* first — palette, typography, components. Critique → refine → lock the system.
3. **Only then touch a portfolio screen.** No full-page generation until the system is locked.
4. Honor the two capstone invariants in every pass: (a) no contact details anywhere;
   (b) footer ends with the Arca logo + "Made for Arca.ph" linked to https://arca.ph.

### The 25-minute exercise that produces this file
- **Step 1 (5m):** Cross-medium warmup — finish the "If my portfolio were a ___" sentence
  three ways, pick the most *me*. → Section 1.
- **Step 2 (5m):** Name ONE non-web inspiration (specific, not "clean/minimal"). → Section 2.
- **Step 3 (10m):** Save 2–3 non-web visual references (NOT portfolios). → `inspiration/references/`.
- **Step 4 (5m):** Write the "never use" list (≥5 items). → Section 5.
- **Step 5 (2m):** Voice (pick one tone) + tagline (ship-tonight version). → Section 6.

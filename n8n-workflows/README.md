# n8n workflow files

Downloaded n8n workflow JSON exports live here.

This folder is part of the project (so it's version-controlled and easy to find),
but it sits OUTSIDE `public/`, so nothing here is published on the live website.

**How to use:** drop your exported `.json` files in this folder. Then just tell me
the file name and I can open and read any of them, to review the workflow, explain
what it does, help document it, or write it up for the portfolio.

Optional: keep the names matching the screenshots for easy pairing, e.g.
`smart-lead-sorter.json` next to `public/images/automations/n8n/smart-lead-sorter.png`.

Export from n8n: open the workflow, then the `...` menu (top right) > Download.
